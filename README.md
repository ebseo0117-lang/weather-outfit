# 날씨 기반 옷 조합 추천 앱
> 오늘 날씨에 맞는 옷차림(상의/하의/신발)을 자동으로 추천해주는 웹앱

## 무엇을 하는 프로젝트인가
매일 아침 옷 고르기 귀찮은 사람들을 위해, 현재 기온과 날씨 상태를 기반으로 어울리는 옷 조합을 카드 형태로 추천한다.

## 사용 방법
1. index.html 파일을 브라우저로 연다.
2. 화면 상단에 현재 날씨와 기온이 표시된다.
3. 기온에 맞는 상의/하의/신발 추천 카드가 자동으로 나타난다.
4. 새로고침 버튼을 누르면 날씨가 다시 랜덤으로 갱신된다.
5. '다른 조합 추천' 버튼을 누르면 같은 기온대의 다른 코디를 볼 수 있다.

## 기온 구간별 추천 기준
- 5도 이하: 두꺼운 긴팔 + 면바지 + 운동화
- 6~14도: 긴팔티 + 청바지 + 운동화
- 15~19도: 셔츠 + 청바지 + 운동화
- 20~24도: 반팔티 + 면바지 + 운동화
- 25도 이상: 반팔티 + 반바지 + 운동화
- 비 오는 날: 신발이 방수 신발로 자동 변경
