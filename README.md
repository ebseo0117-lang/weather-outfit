# 날씨 기반 옷 조합 추천 앱

> 오늘 날씨와 내 취향에 맞는 옷차림(상의/하의/신발)을 추천해주는 웹앱입니다.

## 무엇을 하는 프로젝트인가
매일 아침 옷 고르기 귀찮은 사람들을 위해, 기온·날씨·체질·스타일·TPO(상황)에 맞는 옷 조합을 추천합니다. 내 옷장에 등록한 옷과 색상이 맞으면 매칭 정보도 보여줍니다.

## 사용 방법
1. `index.html` 파일을 브라우저로 엽니다.
2. 상단 슬라이더로 기온을 직접 설정하고, 날씨(맑음/흐림/비/눈)를 선택합니다.
3. 내 체질(추위/보통/더위)을 선택하면 체감 기온이 보정되어 추천이 바뀝니다.
4. 외출 시간대(현재/아침/낮/저녁)를 선택하면 시간대별 기온 보정이 적용됩니다.
5. 스타일(캐주얼/댄디/스포티)과 TPO(출근/데이트/운동/일상)를 선택하면 그에 맞는 코디가 추천됩니다.
6. 추천 카드에 표시되는 색상과 내 옷장에 등록한 옷의 색상이 맞으면 "💡 내 옷장 매칭"이 표시됩니다.
7. 새로고침 버튼으로 날씨를 다시 설정하거나, '다른 조합 추천'으로 같은 조건 내 다른 코디를 볼 수 있습니다.
8. 마음에 드는 코디는 "이 코디 저장하기"로 보관함에 저장할 수 있습니다.
9. 우측 상단 🌙/☀️ 버튼으로 다크모드를 켤 수 있습니다.
10. 우측 상단 ⚙️ 설정에서 OpenWeatherMap API 키를 입력하고 GPS 조회 버튼을 누르면 실시간 위치 기반 날씨를 불러올 수 있습니다.

## 주요 기능
- 기온/날씨 수동 설정 (슬라이더 + 날씨 토글)
- GPS + OpenWeatherMap API 연동으로 실시간 위치 기반 날씨 조회 (선택 사항, 본인 API 키 필요)
- 체질별 체감 온도 보정 (추위 많이 탐 / 보통 / 더위 많이 탐)
- 외출 시간대별 기온 보정 (현재/아침/낮/저녁)
- 스타일 필터 (캐주얼/댄디/스포티)
- TPO 상황 필터 (출근/데이트/운동/일상)
- 비 오는 날 신발 자동 → 방수 신발로 변경
- 날씨별 배경색 및 시각 효과(비/눈/햇살/구름)
- 내 옷장 등록 (이름 + 색상) 및 색상 매칭 추천
- 코디 저장/삭제 (보관함)
- 다크모드 토글 (새로고침 후에도 유지)

## 사용 예시
- 기온 7°C 설정, 체질 '추위 많이 탐' 선택 → 체감 기온 4°C로 보정되어 5도 이하용 코디(두꺼운 맨투맨 등) 추천
- 스타일 '댄디' + TPO '데이트' 선택 → 단정한 셔츠/슬랙스 + 톤온톤 색상 추천
- 옷장에 '베이지 린넨 자켓' 등록 후, 추천 색상이 베이지일 때 "💡 내 옷장 매칭: 베이지 린넨 자켓" 표시
- GPS 조회 클릭 → 현재 위치 기반 실제 기온/날씨로 추천 갱신

## 입력 / 출력

| 입력 | 출력 |
|---|---|
| 기온 슬라이더 / 날씨 토글 | 추천 코디 즉시 갱신, 배경/이펙트 변경 |
| GPS 조회 | 실제 위치 기반 기온/날씨로 갱신 |
| 체질 선택 | 체감 기온 보정 및 추천 구간 변경 |
| 외출 시간대 선택 | 시간대별 보정 기온으로 추천 변경 |
| 스타일 / TPO 선택 | 추천 코디 멘트 및 의상 변경 |
| '이 코디 저장하기' | 보관함에 코디 추가 (TPO/색상 포함) |
| 내 옷장 등록 | 색상 매칭 시 추천 카드에 매칭 정보 표시 |

## 아직 안 되는 것 / 한계
- 외출 시간대별 보정은 실제 기상 예보가 아닌, 입력 기온 기준 가상 보정값입니다.
- GPS/실시간 날씨 API(OpenWeatherMap)는 본인 API 키를 직접 발급받아 입력해야 동작하며, 키 발급 후 활성화까지 시간이 걸릴 수 있습니다.
- 내 옷장 매칭은 색상/카테고리 기준의 단순 매칭이며, 정교한 스타일링 추천은 아닙니다.
